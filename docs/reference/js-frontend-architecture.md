# JS Frontend Architecture

Per-module map of the Flavor Agent editor and admin JavaScript, extracted from the Architecture section of `CLAUDE.md`. Everything under `src/` is built with `@wordpress/scripts` into three webpack entry points — `src/index.js` (editor), `src/admin/settings-page.js` (settings page), and `src/admin/activity-log.js` (AI Activity admin page). For the cross-cutting store, shared-UI, and context internals in depth, see [shared-internals.md](shared-internals.md).
